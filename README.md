# Income-Driven Repayment Simulator

A responsive, lightweight SPA for estimating loan repayment under income-driven repayment (IDR) plans. This is a local-host only application.

## Getting Started

1. **Download** the project ZIP file.
2. **Extract** the contents to a local directory.
3. **Open** the `index.html` file by copying its full file path into your browser's address bar.

## Features

* **Plan and PSLF Support:** Supports the three main plans (RAP, IBR Pre-2014, IBR 2014+) as well as Public Service Loan Forgiveness (PSLF). 
* **Multi-User Simulation:** Input both self and spouse information to simulate how multiple borrowers affect each other's repayment. 
* **Overpayment Support:** Model additional payments to see how they impact total interest and repayment speed.
* **Mid-repayment Support:** Input existing qualified payments or accrued interest to simulate the remainder of your repayment term.
* **Light & Dark Modes:** Toggle between themes for a more comfortable viewing experience.

## Privacy & Storage

* **Local Session Saving:** Your data is saved locally to your browser's `localStorage` and can be encrypted with a user-defined passkey.
* **Persistence:** Be aware that some browsers, such as **Google Chrome**, may retain this local storage data longer than others.

## Disclaimer

Projections are based on 2026 poverty line guidelines and IDR plans. Actual results may vary due to policy changes and life events. This tool does not support private student loans and is not a replacement for professional financial or legal guidance.
